# Sistema-Peluqueria-Canina

Sistema CRUD para gestión de mascotas desarrollado en Java Swing utilizando JPA, Maven y MySQL.

## Tecnologías utilizadas

- Java
- Swing
- JPA
- MySQL
- Maven
- Git/GitHub

---

## Funcionalidades

- Registrar mascotas
- Visualizar datos
- Modificar información
- Eliminar registros
- Persistencia con JPA
